# Fullstack React-MySQL Auth Server

A robust Node.js backend built with Express and MySQL, featuring environment variable protection and a clean API structure for user management. This server is optimized for deployment on **Render**.

## 🚀 Features

- **Express.js Framework**: Fast, unopinionated, minimalist web framework.
- **MySQL Integration**: Persistent data storage using `mysql2` with connection pooling.
- **Security**: Environment variables managed via `dotenv` to keep credentials safe.
- **CORS Enabled**: Ready to communicate with your React frontend.
- **Deployment Ready**: Configured for seamless hosting on Render.

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express
- **Database:** MySQL
- **Environment Management:** Dotenv
- **Development Tools:** Nodemon
